Attributes

Some tags, like the img tag use attributes to define the value of what they're presenting (img src="where the image is located")
The href attribute of the anchor tag, tells you where the anchor leads.
You can also use these attributes to identify your elements

    2 common ways to identify an element using attributes
  • ID - unique way to identify an element
  • Class - a way to identify multiple elements
You need to identify them to able to know where to apply certain rules in CSS or behavior in JS. Then again, you can also use the style attribute to apply style to your elements.

CSS

We use CSS to apply styles to webpages

Ways to include CSS in your HTML

  1. Inline
    • using the style attribute
  2. Internal
    • uses style tag found in head
  3. External
    • Uses a totally separate .css file
    • referenced in head
    • Best practice for separation of concerns

Types of Selectors

Simple Selectors

  • Tag/Element
    • uses the tag name to identify the element
    • this will affect all elements with that tag name
  • Class
    • uses the class name to style a group of elements
    • denoted by .className
  • ID
    • Uses a unique id
    • denoted by #id
  • Pseudo-class
    • specific keyword at the end of the selector used to style elements when it is at a certain state
    • :active, :hover, etc....
  • Pseudo-element
    • used to style specific parts of an element
    • insert content before or after an element
    • style first letter or line of an element
    • ::before, ::after, ::firstline, ::firstletter

Combinator Selectors

  • Child Descendant
  • Sibling
  • Attribute selectors
  • Multiple selectors

Wild card selector

  • Just the *
  • Styles everything, all elements with the given rules
